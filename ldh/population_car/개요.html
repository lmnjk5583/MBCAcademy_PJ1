서울 자치구 교통 CCTV와 자동차 등록 현황 분석

본 프로젝트는 배운 지 약 1개월 시점에서 수행한 5일 미니 프로젝트로, 실제 공공데이터를 활용한 분석 경험을 목표로 한다.

1. 프로젝트 목적

  • 서울 자치구별 교통 관련 CCTV 분포 파악
  • 자치구별 자동차 등록 대수 현황 비교
  • 차량 규모 대비 교통 관리 인프라의 상대적 분포 분석

2. 데이터 선정 이유

서울시는 교통, CCTV, 자동차 관련 공공데이터가 가장 풍부하고 정제되어 있어 단기간 프로젝트에 적합한 분석 대상이다.
  • 서울 열린데이터 광장 제공 데이터 활용
  • 자치구 단위 비교가 가능
  • 현실적인 행정·교통 분석 사례 경험

3. 사용 데이터

  • 서울 자치구별 교통 관련 CCTV 개수
  • 서울 자치구별 자동차 등록 대수

두 데이터를 자치구 기준으로 병합하여 분석을 수행하였다.

4. 분석 방법

  • 자치구별 CCTV 개수와 자동차 등록 대수 비교
  • 단순 수치 비교가 아닌 파생 지표 생성

주요 파생 지표

  • 자동차 1만 대당 CCTV 개수
  • 자치구별 자동차 등록 비중
  • CCTV 밀도 기준 자치구 순위

5. 분석 시 유의점

  • 본 분석은 인과관계 분석이 아님
  • 사고 건수, 교통량 등 추가 데이터는 포함되지 않음
  • 단일 시점 데이터로 추세 분석에는 한계 존재

6. 기대 효과 및 활용

  • 자치구 간 교통 관리 인프라 분포 차이 파악
  • 상대적으로 관리 인프라가 부족한 지역 식별
  • 향후 사고 데이터, 교통량 데이터 결합 분석의 기초 자료

7. 프로젝트 한계 및 향후 개선 방향

  • 교통사고 데이터 결합을 통한 심화 분석 필요
  • 연도별 데이터 확보 시 추세 분석 가능
  • 인구·면적 등 보정 지표 추가 고려
